Tema 2 | Grundlæggende Web

Opgaven

I forløbet grundlæggende web blev vi for første gang introduceret for to af internettets tre-enighed, CSS og HTML. Sammen med en uddelt stilart skulle der researches og laves moodboard og mock-up i Adobe XD. I vores tilfælde fik vi tildelt stilarten, Modern futuristic webdesign - 3D and fluorescent colors.

Ved brug af CSS og HTML lød opgaven på at bygge et simpelt website ud fra et udleveret wireframe og layout diagram. Vores tilgang til opgave var mobile first og derefter lave en desktop versionen ved hjælp af mediaqueries.

Arbejdet

Ud fra vores tildelte stil skulle vi udarbejde et styletile, hvor vi bestemte os for fonte, farver, knap stile, link udseende osv. sammen med et splashbillede. Splashbilledet til websitet skulle vi selv producere ved hjælp af photoshop.

Til websitet skulle der også findes og arbejdes med indhold, billeder og tekst, hvor tekst antal og længde var prædefineret. Udover de førnævnte begrænsninger så var der ellers frie tøjler til at finde på websitets indhold. Som sidste del af opgaven skulle vi få det hele til at spille sammen, farver, fonte og alt andet hvad mit styletile bød på, og så meget responsivitet som muligt.
